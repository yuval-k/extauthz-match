Version 1.0.0

ExtAuth Match is a mobile-friendly authorization interface for agentgateway's external authorization (ext_authz) support. Review and approve/deny API requests in real-time with a swipe interface.

How It Works

When a client makes a request through your agentgateway proxy, it appears as a card in this interface. You can:

  • Swipe right (or click ‚úì) to approve
  • Swipe left (or click ‚úó) to deny

Keyboard Shortcuts

Approve request Y or V
Deny request X

Security

All communication is end-to-end encrypted using AES-256-GCM. The encryption key is in your URL fragment and never leaves your device.

Troubleshooting

If you see "Disconnected":

  • Check that your authorization server is running
  • Verify the URL matches your server's tenant ID
  • Ensure you have network connectivity
